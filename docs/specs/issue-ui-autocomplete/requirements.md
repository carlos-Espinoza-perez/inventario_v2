# Requerimientos (EARS)

## Requerimientos
1. [Req-01] El sistema deberá reemplazar el input de texto simple de búsqueda de producto en `WarehouseEntryScreen` por un componente de autocompletado (select/SearchAnchor).
2. [Req-02] Cuando el usuario haga foco en el componente, el sistema deberá mostrar la interfaz de búsqueda en la parte superior de la pantalla de forma intuitiva, desplegando los resultados en tiempo real.
3. [Req-03] Mientras el usuario escribe un código o nombre, el sistema deberá buscar en la base de datos local (Drift) los productos por similitud.
4. [Req-04] Cuando el usuario seleccione un producto de la lista, el sistema deberá procesar el producto escaneado/seleccionado como si se hubiera introducido un código válido.
