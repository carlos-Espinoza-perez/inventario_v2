MAPEO DE LÓGICA: DE VOZ A CASOS DE USO

1. EXTRACCIÓN ESTRUCTURADA (JSON):
   El modelo GPT-4o-mini procesa el texto y genera una estructura como:
   {
     "action": "ADD_TO_TEMP_LIST",
     "items": [
       {"name": "Camisa Nike", "qty": 15, "price": 15.0, "size": "M"}
     ]
   }

2. BÚSQUEDA SEMÁNTICA (Matching):
   - La IA recibe una lista de SKUs/Nombres disponibles como contexto.
   - Si el usuario dice "Nike", la IA elige el SKU que más se parece basándose en la base de datos actual.

3. ESTADO TEMPORAL (Borrador):
   - Se utiliza un StateNotifier o ChangeNotifier en Flutter para mantener la 'Lista del Secretario'.
   - Esta lista NO toca la base de datos hasta que el usuario da el comando final.

4. INTEGRACIÓN CON USE CASES:
   - Al finalizar, la lista temporal se pasa como argumento al 'RegistrarEntradaUseCase' existente.
   - Esto garantiza que todas las validaciones de bodega, stock y usuario se mantengan intactas.
