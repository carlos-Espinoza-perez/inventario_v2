SISTEMA DE ASISTENTE DE VOZ "SECRETARIO" (MODO MANOS LIBRES - ONLINE)

OBJETIVO:
Permitir la carga masiva de inventario mediante lenguaje natural, donde el 
usuario solo habla y el asistente procesa, confirma y organiza la información 
en tiempo real utilizando modelos de nube para máxima precisión.

COMPONENTES DEL FLUJO:
1. CAPTURA (Oído): Micrófono en modo "Stream" o ráfagas (audio chunks).
2. TRANSCRIPCIÓN (Escritura): OpenAI Whisper API (Conversión de audio a texto de alta fidelidad).
3. RAZONAMIENTO (Cerebro): GPT-4o-mini con "Function Calling" (Extrae datos estructurados).
4. CONTEXTO (Memoria): Inyección de la lista de productos actuales (RAG ligero) para matching preciso de SKUs.
5. RESPUESTA (Voz): OpenAI TTS o Flutter TTS (Confirmación auditiva inmediata).

INFRAESTRUCTURA:
- Cliente: Flutter (Gestión de audio y estado local).
- Backend/Edge: OpenAI API (Procesamiento de lenguaje).
- Base de Datos: Drift local (Persistencia final mediante Use Cases).
